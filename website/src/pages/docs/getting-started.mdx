# Getting started

Before getting started, you need to install the [PromptSnippets](https://chromewebstore.google.com/detail/promptsnippets/dkafkdphnlodjiggkpbokmponlkjfaco) extension.

<Steps>

### Open any website which you want to input some texts

e.g. [chat.openai.com](https://chat.openai.com/)

### Click the extension icon

You can click the extension icon to open/close the extension management panel.

<Image s="guide-1" width={536} height={325}></Image>

### Create a new prompt

Click the `Create` button to create a new prompt.

Every snippets has a `Prefix` and a `Content`.

<Image s="guide-2" width={338} height={255}></Image>

You can use the `prefix` to select the snippet quickly. The `Content` will be inserted into the input box when you select the snippet.

### Use the snippet

Type `/` in any input box, then the snippets panel will appear.

<Image s="guide-3" width={437} height={130}></Image>

You can select a snippet by typing prefix, using arrow keys or mouse, then press `Enter` to insert it.

<Image s="guide-4" width={437} height={227}></Image>

</Steps>

Learn how to use [variable snippets](/docs/create-snippets) to get more powerful features.
